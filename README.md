## 🚀 Features

✨ Designed for both **creators** and **developers**

* 🎯 **Instant Thumbnail Extraction**
  `Fetch all available YouTube thumbnail resolutions in one click`

* 🧠 **Smart Detection Engine**
  `Filters invalid placeholders and returns only valid images`

* 👀 **Live Preview System**
  `Embedded player dynamically updates with video ID`

* ⚡ **Lightning Fast UI**
  `Optimized for low-end devices with buttery smooth performance`

* 🎬 **GSAP Animations**
  `Scroll-triggered storytelling + fluid transitions`

* 🌗 **Dark / Light Mode**
  `Seamless theme switching with persistent memory`

* 📦 **Ready-to-Download Assets**
  `Auto-labeled files by quality (HD, HQ, MQ, etc.)`

## ⚙️ Installation

## Clone the repository
    git clone https://github.com/SaeedX302/TSun-Youtube-Thumbnail-Downloader.git

## Navigate into the project
    cd TSun-Youtube-Thumbnail-Downloader

## Run a local server 
    python -m http.server 3000
    npx serve 
---

## 🧑‍💻 Usage

```text
1. Paste YouTube URL or Video ID
2. Click "Generate"
3. Preview thumbnails instantly
4. Download your desired resolution
```

💡 Pro Tip:
Press **Enter** instead of clicking for faster workflow ⚡
