# Contributing to TSun Youtube Thumbnail Downloader

First off, thank you for considering contributing to this project! Even though the source code is copyrighted and proprietary (meaning nobody is allowed to take credit for or re-upload this project as their own), **contributions, improvements, and bug fixes from the community are warmly welcomed**.

## How Can I Contribute?

### Reporting Bugs
If you find a bug in the source code, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.

### Implementing Features
We always welcome new features or UI improvements! To submit one:
1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
4. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
5. Push to the branch (`git push origin feature/AmazingFeature`).
6. Open a Pull Request from your fork to the main repository.

## Important Note on Contributions
By submitting a pull request, you acknowledge and agree that:
1. Your contributions will be merged into the project under the existing proprietary license held by **SaeedX302**.
2. You agree not to claim sole ownership of the project.
3. You respect the original creator's credits and will not attempt to redistribute the whole project under your name.

Once again, thank you for your help in making this tool better for everyone!
